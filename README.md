# CX2118 AI Gallery

AI 生成图片分享平台

## 如何添加内容

### 1. 新建文件夹
在仓库根目录创建新文件夹，以角色/主题命名，例如 `小鸟游星野`

### 2. 添加图片
将 PNG/JPG 图片放入文件夹中

### 3. 添加描述文件
在文件夹内创建 `photo.md`，写入该文件夹的简介：

```markdown
小鸟游星野
```

### 4. 更新根目录索引
编辑根目录的 `photo.md`，添加新卡片：

```markdown
# AI Gallery

## 小鸟游星野
小鸟游星野

## 新角色名
新角色的简介
```

### 5. 等待部署
推送到 main 分支后，GitHub Pages 会自动更新（约1-2分钟）
